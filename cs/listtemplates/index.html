cs listTemplates

List all public, private, and privileged templates.

Flags

– --templatefilter value - [required] possible values are «featured», «self», «selfexecutable»,«sharedexecutable»,«executable», and «community». * featured : templates that have been marked as featured and public. * self : templates that have been registered or created by the calling user. * selfexecutable : same as self, but only returns templates that can be used to deploy a new VM. * sharedexecutable : templates ready to be deployed that have been granted to the calling user by another user. * executable : templates that are owned by the calling user, or public templates, that can be used to deploy a VM. * community : templates that have been marked as public but not featured. * all : all templates (only usable by admins).
– --account value - list resources by account. Must be used with the domainId parameter.
– --domainid value - list only resources belonging to the domain specified
– --hypervisor value - the hypervisor for which to restrict the search
– --id value - the template ID
– --isrecursive value - defaults to false, but if true, lists all resources from the parent specified by the domainId till leaves. (default: unset)
– --keyword value - List by keyword
– --listall value - If set to false, list only resources belonging to the command’s caller; if set to true - list resources that the caller is authorized to see. Default value is false (default: unset)
