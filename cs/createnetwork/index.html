cs createNetwork

Creates a network

Flags

– --account value - account who will own the network
– --aclid value - Network ACL Id associated for the network
– --acltype value - Access control type; supported values are account and domain. In 3.0 all shared networks should have aclType=Domain, and all Isolated networks - Account. Account means that only the account owner can use the network, domain - all accouns in the domain can use the network
– --displaynetwork value - an optional field, whether to the display the network to the end user or not. (default: unset)
– --displaytext value - [required] the display text of the network
– --domainid value - domain ID of the account owning a network
– --endip value - the ending IP address in the network IP range. If not specified, will be defaulted to startIP
– --endipv6 value - the ending IPv6 address in the IPv6 network range
– --gateway value - the gateway of the network. Required for Shared networks and Isolated networks when it belongs to VPC
– --ip6cidr value - the CIDR of IPv6 network, must be at least /64
– --ip6gateway value - the gateway of the IPv6 network. Required for Shared networks and Isolated networks when it belongs to VPC
– --isolatedpvlan value - the isolated private vlan for this network
– --name value - [required] the name of the network
– --netmask value - the netmask of the network. Required for Shared networks and Isolated networks when it belongs to VPC
– --networkdomain value - network domain
– --networkofferingid value - [required] the network offering id
– --physicalnetworkid value - the Physical Network ID the network belongs to
– --startip value - the beginning IP address in the network IP range
– --startipv6 value - the beginning IPv6 address in the IPv6 network range
– --subdomainaccess value - Defines whether to allow subdomains to use networks dedicated to their parent domain(s). Should be used with aclType=Domain, defaulted to allow.subdomain.network.access global config if not specified (default: unset)
– --vlan value - the ID or VID of the network
– --zoneid value - [required] the Zone ID for the network
– --debug, -d -
– --dry-run, -D -
– --region value, -r value -
